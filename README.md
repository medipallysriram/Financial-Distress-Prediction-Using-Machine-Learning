# 📊 Hybrid Network Analysis and Machine Learning for Financial Distress Prediction

Hybrid Network Analysis and Machine Learning for Financial Distress Prediction is a data science project that combines financial indicators and network-based relationships to identify companies at risk of financial distress. The system leverages machine learning algorithms and graph analytics techniques to improve prediction accuracy and support early financial risk detection.

## 📌 Features

* Financial distress prediction
* Financial ratio analysis
* Network-based feature extraction
* Graph and relationship analysis
* Machine learning classification models
* Data preprocessing and visualization
* Risk assessment and prediction reports
* Performance evaluation using accuracy metrics

## 🛠 Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* XGBoost
* NetworkX
* Matplotlib
* Jupyter Notebook

## 🎯 Project Objective

The objective of this project is to develop an intelligent financial distress prediction system by integrating network analysis and machine learning techniques. The model helps identify financially vulnerable companies at an early stage, enabling better risk management and decision-making for investors, financial institutions, and businesses.
